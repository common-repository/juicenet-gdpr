== Description ==
This plugin show Privacy policy and cookies policy for these website service:

ESSENTIAL COOKIES (THIS WEBSITE)
CONTACT FORM (THIS WEBSITE)
DIRECT REGISTRATION (THIS WEBSITE)
GOOGLE RECAPTCHA (GOOGLE INC.)
MAILCHIMP (THE ROCKET SCIENCE GROUP, LLC.)
TWITTER TWEET BUTTON AND SOCIAL WIDGETS (TWITTER, INC.)
FACEBOOK LIKE BUTTON AND SOCIAL WIDGETS (FACEBOOK, INC.)
MAILCHIMP WIDGET (THE ROCKET SCIENCE GROUP, LLC.)
YOUTUBE VIDEO WIDGET (GOOGLE INC.)
INSTAGRAM WIDGET (INSTAGRAM, INC.)
Widget Google Maps (Google Inc.)
Widget Video YouTube (Google Inc.)

The plugin show an OPT IN banner on the first access to your website.
This plugin is implemented for Juice customer websites, but if you want to use it, it's not a problem.

== Installation ==

1. Upload `juicenet-gdpr` dir to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add your personal data to Juice GDPR options page and save.
1. Place the generate shortcode in your page or template.
